Q-music 0.2.0

本版本基于 LX Music Desktop 2.12.2 二次修改。

### 界面全面焕新（液态玻璃）

- 外壳与内容面板改为低透明度中性白毛玻璃，自定义背景图能真正透过面板显示。
- 新增「背景与毛玻璃」设置：一键选择全局背景图，面板透明 / 模糊程度滑块实时调整并记忆。
- 主题颜色选择器升级为迷你玻璃预览卡，选中态带主题色柔光圈。
- 左侧导航去掉选中竖线，改为主色实心图标胶囊；设置侧栏图标未选中改为中性灰。
- 右键 / 下拉菜单统一为主题感知磨砂浮层，主色柔和 hover 胶囊。
- SQ / Hi-Res 等徽标胶囊化。

### 播放栏重做

- 播放簇（上一首 / 播放 / 下一首）真正居中，图标重绘为实心圆角风格。
- 播放键改为主题色渐变 + 发光；进度条改为主题色，新增 hover / 拖拽圆点。
- 右侧工具图标按「面板开关 | 播放设置 | 动作」分组加细分隔线；当前播放时间加重显示。

### 列表与动画

- 列表行优化：正在播放行主色高亮 + 均衡跳动条，hover 磨砂圆角底，文字加可读性微衬底。
- 打开歌曲评论改用 FLIP 连续滑动动画，卡顿吸收在静止阶段，全程丝滑。

### 更新设置

- 新增「启动时自动检查更新」开关；修复「软件更新」设置节不可见的问题。
